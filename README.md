# oneplus5-nethunter-ath9k-htc
OnePlus 5/5T Kali NetHunter kernel with ath9k_htc support for Alfa AWUS036NHA / AR9271.
## License

This repo is licensed under GPL-2.0-only, same as the Linux kernel.

The kernel source base is LineageOS `android_kernel_oneplus_msm8998`.
My changes are provided as a config + diff so people can rebuild instead of trusting a random boot image.
